# Api Speech Recognition

### Running Locally

1 - create virtual env

```bash
python -m venv venv
```

2 - active virtual env

```bash
source keyword/bin/activate
```

3 - Install packages

```bas
pip install -r requirements.txt
```

4 Run

```bash
python app.py
```

### Running in Docker

1 -  Run docker-compose

```bash
docker-compose up --build
```
